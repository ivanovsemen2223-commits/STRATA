# STRATA — лендинг архитектурного бюро

🔗 Демо: https://apex-arc-theta.vercel.app

![Превью проекта](preview.jpg)

## О проекте

С первого экрана сайт продаёт ощущение премиальной студии. Полноэкранный hero с фото современного дома, заголовок и кнопка-пилюля посажены внизу слева — взгляд скользит по фото прямо к «Записаться на консультацию». Светлая палитра, много воздуха и крупная типографика Manrope — визуальный язык, который считывает аудитория с бюджетом на дом или интерьер.

Дальше страница ведёт клиента по воронке: цифры доверия (25+ лет, 500+ проектов, 98% удержания, 15+ стран) → четыре услуги крупными карточками (архитектура, интерьер, градостроение, управление) → портфолио реальных типов объектов (дом, отель, офис, ресторан) → отзывы, которые закрывают возражение «а вы справитесь?». Кнопка в подвале ведёт в Telegram — один понятный шаг к контакту.

## Как это работает

Проект сознательно лёгкий — почти весь эффект достигается вёрсткой, а не скриптами:

- **Раскладка** — на чистом **CSS Grid**. Hero собран сеткой так, чтобы заголовок и кнопка стояли внизу-слева, а боковое фото тянулось вдоль обоих рядов.
- **Появление блоков** — `script.js` навешивает `IntersectionObserver`: при попадании в зону видимости элементы `.reveal` мягко выезжают (класс `.in`) с лёгким стаггером по четвёркам. Без библиотек.
- **Контакт** — кнопки и футер ведут в Telegram (@Frilste).

## Стек

HTML / CSS / нативный JS · раскладка на **CSS Grid** · reveal на **IntersectionObserver** · шрифты **Manrope / Inter**. Сборка не нужна — чистая статика, поэтому быстро грузится и выглядит дорого на любом экране.

